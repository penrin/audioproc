# AudioProc


概要
-----

- 音響計測などで使う Python3 パッケージ


インストール
----------

- インストール

```
$ sudo pip install git+https://github.com/penrin/audioproc.git
```

- アップデート

```
$ sudo pip install -U git+https://github.com/penrin/audioproc.git
```

- アンインストール

```
$ sudo pip uninstall audioproc
```
